# Social Login UI Demo

A polished, responsive login screen built with **HTML, CSS, and a tiny bit of JavaScript**. This project is designed for portfolio use and front-end layout practice.

> **Ethical note:** This is a non-functional UI demo. It does not collect, store, or submit login credentials.

## Features

- Mobile-first responsive layout
- Clean card-based interface
- Accessible labels and semantic HTML
- CSS custom properties for easy theme edits
- Focus states and reduced-motion support
- GitHub Pages ready

## Getting Started

Open `index.html` in your browser.

## Deploy on GitHub Pages

1. Upload this project folder to a GitHub repository.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/root` folder.
5. Save and open the generated GitHub Pages link.
